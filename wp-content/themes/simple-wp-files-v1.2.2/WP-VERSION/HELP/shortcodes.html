Intro - top

This theme comes with several built in shortcodes. In order to render them correctly you must write them inside the post/page content area as shown in the picture below.

Pre - top

This shortcode will render preformated text. Useful for text code examples.

[pre]content[/pre]

Link buttons - top

Render links as buttons

[link_button href="link" target="_self|_blank"]text[/link_button]

Tabs - top

Organize your content inside tabbed panes

[tabs]
	  [tab title="your title"]...[/tab]
	  [tab title="your title"]...[/tab]
	  ...
[/tabs]

[one_half]...[/one_half] [one_half_last]...[/one_half_last]

 

Three column layout - top
